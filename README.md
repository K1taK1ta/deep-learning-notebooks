# deep-learning-notebooks


## Sequence classification. RNN
Классификация эмоций в тексте ([GoEmotions](https://huggingface.co/datasets/google-research-datasets/go_emotions), raw) с RNN (GRU / LSTM)

## Seq2Seq. RNN
Машинный перевод RU → EN на Seq2Seq (RNN encoder-decoder), датасет Tatoeba


## Caltech101. CNN
Задача классификации изображений на классическом датасете Caltech101


## Bipedal Walker. RL
Проект посвящен обучению агента ходьбе в непрерывной среде BipedalWalker-v3 (hardcore=False) с использованием самостоятельно реализованного алгоритма DDPG (Deep Deterministic Policy Gradient)


## Image Captioning
Проект посвящен задаче Image Captioning — автоматической генерации подписей к изображениям. Модель обучается на популярном датасете MS COCO 2017 с обязательным использованием не менее 25% всей тренировочной выборки для настройки нейросети, генерирующей точные текстовые описания к картинкам


## Image Classification via Transfer Learning
Проект посвящен решению задачи классификации изображений с применением технологии трансферного обучения (Transfer Learning)
